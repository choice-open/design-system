# Choice UI

A desktop-first UI component library built for professional desktop applications.

## Getting Started

Visit https://choice-ui.com/guide to get started with Choice UI.

## Documentation

Visit https://choice-ui.com/docs to view the full documentation.

## Storybook

Visit https://storybook.choice-ui.com to view the storybook for all components.

## Installation

```bash
pnpm add @choice-ui/react @choice-ui/design-tokens
# or
npm install @choice-ui/react @choice-ui/design-tokens
```
