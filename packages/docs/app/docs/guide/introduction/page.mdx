# Introduction

Welcome to the **Hero UI Docs**. This documentation is built with Next.js App Router and MDX.

## Features

- **Next.js 15+**: App Router, Server Components.
- **Tailwind CSS 4**: The latest utility-first CSS framework.
- **MDX**: Write content in Markdown with React components.
- **Dark Mode**: Built-in support for light and dark themes.
